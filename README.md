# Copy-Compare 

Copy&amp;Compare is an interactive android game that tests user drawing skills by showing them a picture and making them copy it as closely as possible.
<br> The user will then receive stars for their drawing. If the user receive 2 or more star. They passed.
<br>The game comes in 2 mode - 'easy' and 'hard' and each mode has 3 different level, as follow: 
<br>Level 1: Draw a circle <br>
Level 2: Draw a tree <br>
Level 3: Draw a cat <br>
